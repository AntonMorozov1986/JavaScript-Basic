Упражнение №4. Атрибуты ASYNC и DEFER тега SCRIPT.

Атрибуты ASYNC и DEFER задают порядлк обработки внешнего файла JavaScript интерпретатором. По умолчанию эти атрибуты отключены и интерпретатор HTML дойдя до тега SCRIPT приостановит обработку страницы и будет загружать внешний файл JS и потом его обрабатывать. После обработки файла JS интерпретатор HTML продолжит обработку страницы. Атрибуты ASYNC и DEFER позволяют изменить порядок обработки файла JS. При наличии атрибута ASYNC загрузка файла будет происходить паралельно с обработкой страницы и приостановится для выполнения скрипта только после его загрузки. При наличии атрибута DEFER загрузка файла будет происходить паралельно с обработкой страницы, а обработка файла JS произойдет только полной обработки страницы. Атрибуты ASYNC и DEFER будут работать только при загрузке скрипта с внешнего файла, т.е. при наличии атрибута SRC. Если задано сразу два атрибута — ASYNC и DEFER, то атрибут ASYNC игнорируется.
